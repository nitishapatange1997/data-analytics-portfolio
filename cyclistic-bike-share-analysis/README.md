# 🚴 Cyclistic Bike-Share Analysis

## 📌 Overview

This project analyzes bike-share data to understand behavioral differences between casual riders and annual members, with the goal of increasing membership conversion.

---

## 🎯 Business Problem

How do casual riders and annual members use Cyclistic bikes differently?

---

## 🛠 Tools & Technologies

* SQL (Google BigQuery)
* Tableau
* Google Cloud Storage

## 🔍 Data Processing Steps

* Combined 12 months of trip data using SQL
* Created new features (ride length, day of week)
* Cleaned data by removing invalid durations and missing values
* Validated geographic coordinates

---

## 📈 Key Insights

* Members account for ~64% of total rides
* Casual riders have significantly longer ride durations
* Members ride more frequently on weekdays
* Casual riders are more active on weekends

---

## 💡 Recommendations

* Target casual riders during peak weekend usage
* Highlight cost benefits of annual membership
* Encourage habit formation through incentives
* Use lifestyle-based marketing strategies

---

## 🚀 Conclusion

The analysis reveals clear behavioral differences between casual riders and members. These insights can help Cyclistic design targeted strategies to convert casual riders into long-term members and improve overall profitability.
